# 项目空间页面原型

## 项目空间公共头

所有项目空间页面顶部固定显示：

```text
项目：order-service   分支：main   快照：abc123   项目状态：active
代码分析：partial_failed   索引：stale   Neo4j：ready
快捷操作：[同步仓库] [代码分析] [构建索引] [可信检索] [AI 上下文包]
```

状态异常时显示可点击提示，跳转到项目任务或对应功能页。

## 项目总览

### 页面目标

作为项目空间首页，汇总项目健康度、最近任务、知识和索引状态，并提供高频入口。

### 线框

```text
页面标题：项目总览

指标卡：
  代码事实 12,430    接口 186    知识资产 84    已确认 52
  索引状态 stale     待复审 9    失败任务 2    最近导出 6

主区域：
┌──────────────────────┬──────────────────────┐
│ 最近任务              │ 知识质量              │
│ - 代码分析 partial    │ 缺证据 12             │
│ - 索引构建 failed     │ 待复审 9              │
│ - 上下文包 succeeded  │ 逾期复审 3            │
├──────────────────────┼──────────────────────┤
│ 快捷入口              │ 最近审计              │
│ [代码事实] [知识资产] │ 用户、动作、时间       │
│ [可信检索] [AI助手]   │                      │
└──────────────────────┴──────────────────────┘
```

### 主要操作

- 同步仓库。
- 触发代码分析。
- 构建索引。
- 进入可信检索。
- 进入 AI 研发助手。
- 查看失败任务。

## 项目配置

### 页面目标

配置项目仓库、默认代码路径、知识库路径、索引策略和归档状态。

### 线框

```text
页面标题：项目配置

基础信息
  项目名称 [________]
  描述     [________]
  状态     active

仓库配置
  Git 地址 [________]
  默认分支 [main____]
  默认代码路径 [src___]
  默认知识库路径 [docs_]

索引配置
  来源范围 [代码] [知识文档] [知识资产] [接口] [SQL] [调用链]
  topK 默认值 [10]
  低命中阈值 [0.35]

底部：[保存] [同步仓库] [归档项目]
```

### 状态

| 状态 | 表现 |
|---|---|
| 路径非法 | 字段下方显示错误，不允许保存 |
| Git 不可访问 | 测试连接失败，保留原配置 |
| 项目归档 | 写操作禁用，只允许恢复 |

## 项目成员

### 页面目标

项目负责人维护项目成员和项目角色。

### 线框

```text
页面标题：项目成员                         [添加成员]

┌────────┬────────────┬────────┬────────────┬────────┐
│ 用户名 │ 项目角色    │ 状态   │ 加入时间    │ 操作   │
├────────┼────────────┼────────┼────────────┼────────┤
│ 张三   │ 负责人      │ 启用   │ 2026-06-18 │ 修改   │
│ 李四   │ 成员        │ 启用   │ 2026-06-18 │ 移除   │
└────────┴────────────┴────────┴────────────┴────────┘
```

### 主要操作

- 添加成员。
- 修改项目角色。
- 移除成员。
- 查看只读成员权限说明。

### 校验

- 项目至少保留一个负责人。
- 停用用户不能新增授权。
- 项目成员不能修改成员权限。

## 项目任务

### 页面目标

统一查看仓库同步、代码分析、索引构建、上下文生成和影响分析任务。

### 线框

```text
页面标题：项目任务
筛选：任务类型 [全部] 状态 [全部] 触发人 [全部] 时间 [____]

┌────────────┬────────────┬────────────┬────────┬────────────┬────────┐
│ 任务类型    │ 状态        │ 触发人      │ 耗时   │ 失败原因     │ 操作   │
└────────────┴────────────┴────────────┴────────┴────────────┴────────┘
```

### 操作

- 查看任务详情。
- 重试失败任务。
- 跳转源功能页。
- 查看失败文件或错误日志摘要。

## 项目审计入口

项目总览和项目设置中都提供项目审计入口。完整页面见 `08-治理复审与审计页面.md`。
