功能拆分后的串联方式

页面不是孤岛
组织层
创建项目、分配负责人、配置 MySQL/Qdrant/Neo4j。
项目空间
触发仓库同步、代码分析、索引构建和任务查看。
代码智能
将源码转为 CodeFact，并写入图关系。
知识资产
把文档确认成可检索知识，并绑定证据。
检索上下文
从统一索引召回对象，装配 ContextPackage。

变更影响

候选初判

输入文件列表或 diff，系统结合 CodeFact、Neo4j 有限路径和 Qdrant 语义召回，输出影响接口、知识和 ReviewTask 候选。

治理审计

闭环

低命中、缺证据、复审逾期、变更命中和失败任务都会进入治理看板，处理结果写回知识、索引、图和审计事件。
