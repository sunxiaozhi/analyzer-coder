# 知识资产页面原型

## 知识文档

### 页面目标

维护项目知识库文件，支持 Markdown 预览和模板应用。

### 线框

```text
页面标题：知识文档                         [新建文档] [应用模板]

┌──────────────────┬─────────────────────────────────────────┐
│ 文件树            │ 编辑/预览区                              │
│ docs/             │ Tabs: [编辑] [预览] [元信息]              │
│  - adr/           │                                         │
│  - api/           │ Markdown 编辑器 / 预览                    │
│  - incident/      │                                         │
└──────────────────┴─────────────────────────────────────────┘

底部状态：保存状态、更新时间、创建人
```

### 主要操作

- 新建文件或目录。
- 编辑知识文档。
- 删除文件。
- 应用模板。
- Markdown 预览。
- 从文档内容创建知识资产草稿。

### 状态

| 状态 | 表现 |
|---|---|
| 知识路径未配置 | 空状态提示项目配置知识库路径 |
| 文件冲突 | 保存失败，提示重新加载 |
| 只读成员 | 只显示预览，不显示编辑按钮 |

## 知识资产

### 页面目标

维护结构化知识资产，完成草稿、确认、待复审和归档流转。

### 线框

```text
页面标题：知识资产                         [新建知识]
筛选：类型 [全部] 状态 [全部] 标签 [____] 负责人 [全部] 复审 [全部]

┌──────────────┬────────────┬────────┬────────┬────────────┬────────┐
│ 标题          │ 类型        │ 状态   │ 证据数 │ 复审日期     │ 操作   │
├──────────────┼────────────┼────────┼────────┼────────────┼────────┤
│ 订单取消规则  │ business_rule│ confirmed│ 3    │ 2026-08-01 │ 详情   │
│ 用户接口说明  │ api_doc     │ draft  │ 1      │ -          │ 编辑   │
└──────────────┴────────────┴────────┴────────┴────────────┴────────┘
```

### 主要操作

- 新建知识资产。
- 编辑草稿。
- 绑定 Evidence。
- 项目负责人确认。
- 标记待复审。
- 归档。
- 加入上下文包。

## 知识资产详情抽屉

### 线框

```text
抽屉：知识资产详情

基础信息
  标题、类型、状态、标签、负责人、确认人、确认时间、复审日期

正文
  Markdown 预览

Evidence
  ┌────────────┬────────────┬────────────┬────────┬────────┐
  │ 证据类型    │ 来源对象     │ 文件/符号    │ 置信度 │ 操作   │
  └────────────┴────────────┴────────────┴────────┴────────┘

图关系
  关联 CodeFact、ContextPackage、ReviewTask

操作
  [编辑] [绑定证据] [确认] [标记待复审] [归档] [加入上下文]
```

### 权限

| 角色 | 权限 |
|---|---|
| 项目负责人 | 编辑、确认、标记待复审、归档、绑定证据 |
| 项目成员 | 创建草稿、编辑自己的草稿、绑定证据 |
| 只读成员 | 查看 |

## 证据绑定弹窗

### 页面目标

把代码事实、接口、SQL、检索结果或外部引用绑定为知识证据。

### 线框

```text
弹窗：绑定证据

证据来源 Tab:
  [代码事实] [接口] [SQL] [检索结果] [外部引用]

筛选区 / 搜索区

候选列表
  来源对象、文件路径、符号、摘要、置信度

证据说明 [________________]
置信度   [高/中/低]

[取消] [绑定]
```

### 规则

- 只能绑定当前项目对象。
- 检索结果复用时后端重新校验源对象。
- 绑定成功后标记相关 IndexRecord 为 stale。

## 知识模板

### 页面目标

维护项目级知识模板，供知识文档和知识资产创建时复用。

### 线框

```text
页面标题：知识模板                         [新建模板]

左侧：模板列表
  ADR
  API 说明
  故障复盘
  业务规则

右侧：模板编辑
  模板名称
  适用类型
  Markdown 模板内容

[保存] [删除]
```

### MVP 模板

- ADR。
- API 说明。
- 故障复盘。
- 业务规则。
- 开发规范。
