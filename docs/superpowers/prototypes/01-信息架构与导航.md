# 信息架构与导航原型

## 全局布局

```text
┌──────────────────────────────────────────────────────────────┐
│ 顶栏：平台名称 | 项目选择器 | 全局搜索入口 | 当前用户 | 退出 │
├───────────────┬──────────────────────────────────────────────┤
│ 左侧导航       │ 主内容区                                      │
│               │                                              │
│ 组织层菜单     │ 页面标题 / 状态条 / 操作区                     │
│ 项目空间菜单   │ 页面主体                                      │
│               │                                              │
└───────────────┴──────────────────────────────────────────────┘
```

## 顶栏

| 区域 | 内容 | 行为 |
|---|---|---|
| 平台名称 | 综合研发智能平台 | 点击回到最近访问项目或项目列表 |
| 项目选择器 | 当前项目名称、状态、分支 | 切换项目；无项目时进入项目列表 |
| 全局搜索入口 | 默认进入当前项目可信检索 | 未选项目时禁用 |
| 当前用户 | 用户名、角色、项目角色 | 下拉显示修改密码、退出 |
| 状态提示 | Qdrant、Neo4j、任务失败摘要 | 管理员可跳转系统配置 |

## 组织层导航

仅系统管理员完整可见。普通用户只看到项目列表。

```text
组织层
  - 项目列表
  - 账号与权限
  - 系统配置
```

## 项目空间导航

进入项目后显示。所有项目空间页面必须带 `projectId`。

```text
项目空间

  - 代码智能
    - 代码事实
    - 接口关系
  - 知识资产

    - 知识资产
    - 知识模板
  - 检索与上下文
    - 统一索引
    - 可信检索
    - AI 研发助手
    - 上下文包记录

  - 治理复审

```

## 角色可见性

| 页面组 | 系统管理员 | 项目负责人 | 项目成员 | 只读成员 |
|---|---|---|---|---|
| 项目列表 | 全部项目 | 授权项目 | 授权项目 | 授权项目 |
| 账号与权限 | 可见 | 不可见 | 不可见 | 不可见 |
| 系统配置 | 可见 | 不可见 | 不可见 | 不可见 |
| 项目总览 | 可见 | 可见 | 可见 | 可见 |
| 项目配置 | 可见 | 可编辑 | 只读 | 只读 |
| 项目成员 | 可见 | 可编辑 | 不可见 | 不可见 |
| 代码智能 | 可见 | 可操作 | 可操作 | 只读 |
| 知识资产 | 可见 | 可确认 | 可创建草稿 | 只读 |
| 统一索引 | 可见 | 可构建 | 可构建 | 只读 |
| 可信检索 | 可见 | 可见 | 可见 | 可见 |
| AI 研发助手 | 可见 | 可导出 | 可导出 | 只读 |
| 变更影响 | 可见 | 可操作 | 可操作 | 只读 |
| 治理复审 | 可见 | 可处理 | 只读或不可见 | 只读或不可见 |
| 项目审计 | 可见 | 可见 | 不可见 | 不可见 |

## 页面通用状态

| 状态 | 页面表现 |
|---|---|
| 未登录 | 跳转登录页 |
| 未选择项目 | 主内容区显示项目选择空状态 |
| 无项目权限 | 403 空状态，不展示项目摘要 |
| 项目归档 | 顶部显示归档标签，大部分写操作禁用 |
| 任务运行中 | 顶部状态条显示进度，列表行显示运行中 |
| 部分失败 | 保留成功结果，显示失败原因和失败文件入口 |
| 存储不可用 | 顶部健康提示，相关页面显示降级状态 |

## 跳转关系

```text
项目列表 -> 项目总览
项目总览 -> 代码事实 / 知识资产 / 统一索引 / 可信检索 / AI 研发助手
代码事实详情 -> 绑定证据 / 加入上下文包 / 查看图关系
可信检索结果 -> 加入上下文包 / 绑定证据 / 创建知识草稿 / 跳转源对象
AI 研发助手 -> 上下文包详情 / 导出审计
变更影响结果 -> ReviewTask / 知识资产详情 / 代码事实详情
治理看板 -> ReviewTask / 知识资产详情 / 低命中查询 / 失败任务
项目任务 -> 任务详情 / 失败重试 / 源页面
项目审计 -> 目标对象详情
```
