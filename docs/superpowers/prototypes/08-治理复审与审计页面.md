# 治理复审与审计页面原型

## 治理看板

### 页面目标

汇总项目知识质量、任务失败、索引状态、低命中查询、上下文导出和待复审项。

### 线框

```text
页面标题：治理看板

指标卡：
  待复审 9 | 缺证据 12 | 索引失败 1 | 分析失败 2 | 低命中 18

治理项分组：
┌──────────────────────┬──────────────────────┐
│ 知识质量              │ 任务与索引             │
│ - 缺证据知识          │ - 索引失败             │
│ - 复审逾期            │ - 分析失败             │
├──────────────────────┼──────────────────────┤
│ 使用反馈              │ 变更影响               │
│ - 低命中查询          │ - 命中已确认知识        │
│ - 上下文导出记录       │ - 待复审候选            │
└──────────────────────┴──────────────────────┘
```

### 主要操作

- 跳转知识资产。
- 创建 ReviewTask。
- 触发重建索引。
- 查看失败任务。
- 查看低命中查询。

## ReviewTask 列表

### 页面目标

项目负责人处理复审任务。

### 线框

```text
页面标题：ReviewTask
筛选：状态 [open] 原因 [全部] 负责人 [全部] 到期 [全部]

┌────────────┬────────────┬────────────┬────────┬────────────┬────────┐
│ 关联知识    │ 触发原因     │ 状态        │ 负责人 │ 到期时间     │ 操作   │
└────────────┴────────────┴────────────┴────────┴────────────┴────────┘
```

### 状态

- `open`
- `resolved`
- `ignored`
- `reopened`

## ReviewTask 处理抽屉

### 线框

```text
抽屉：ReviewTask 处理

触发原因
  变更影响命中 / Evidence 失效 / 复审逾期 / 低命中治理

关联对象
  KnowledgeAsset
  Evidence
  ChangeAnalysisTask
  图路径

处理区
  处理结论 [已更新知识 | 确认仍有效 | 暂不处理]
  说明 [________________]

操作
  [打开知识详情] [更新 Evidence] [标记 resolved] [忽略]
```

### 规则

- `resolved` 后回写 KnowledgeAsset 或 Evidence 状态。
- `ignored` 必须填写原因。
- 任一处理结果必须写 AuditEvent。
- 处理后标记索引 stale，并更新 Neo4j 图关系。

## 低命中治理

### 页面目标

查看低命中或无结果查询，判断是否需要补充知识。

### 线框

```text
页面标题：低命中治理
筛选：时间 [近7天] 来源 [可信检索/上下文召回] 处理状态 [全部]

┌────────────┬────────────┬────────┬────────┬────────────┬────────┐
│ 查询摘要    │ 结果数       │ topScore│ 次数   │ 最近时间     │ 操作   │
└────────────┴────────────┴────────┴────────┴────────────┴────────┘
```

### 主要操作

- 查看脱敏查询摘要。
- 重新检索。
- 创建知识草稿。
- 指派 ReviewTask。
- 标记无需处理。

### 安全

SearchLog 只展示脱敏摘要，不展示密码、token、私钥或完整敏感请求体。

## 项目审计

### 页面目标

按操作人、动作、对象和时间查询项目内关键审计事件。

### 线框

```text
页面标题：项目审计
筛选：
  操作人 [全部]
  动作 [全部]
  对象类型 [全部]
  时间 [____ - ____]

┌────────────┬────────┬────────────┬────────────┬────────────┬────────┐
│ 时间        │ 操作人 │ 动作        │ 对象类型     │ 对象摘要     │ 结果   │
└────────────┴────────┴────────────┴────────────┴────────────┴────────┘
```

### 审计动作

- 登录。
- 项目创建。
- 成员授权。
- 项目配置修改。
- 知识确认。
- 知识归档。
- Evidence 绑定。
- 索引构建。
- 上下文包导出。
- 变更影响分析。
- ReviewTask 处理或忽略。

### 规则

- 审计不保存敏感明文。
- 审计记录必须能跳转目标对象；目标已删除或归档时显示历史摘要。
- 只读成员默认不可见项目审计。
