# 检索与上下文页面原型

## 统一索引

### 页面目标

构建和查看当前项目的 Qdrant 统一索引。

### 线框

```text
页面标题：统一索引                         [构建索引]
状态条：状态 stale | 最近成功 2026-06-18 10:20 | 失败 1 次

指标卡：
  总记录 23,450 | 代码 12,300 | 知识 300 | 接口 186 | SQL 920

来源分布图 / 状态分布图

索引记录列表
┌────────────┬────────────┬────────────┬────────┬────────────┬────────┐
│ 来源类型    │ 标题        │ 状态        │ chunk  │ 更新时间     │ 操作   │
└────────────┴────────────┴────────────┴────────┴────────────┴────────┘

失败原因面板
```

### 主要操作

- 全量构建索引。
- 查看索引记录。
- 查看失败原因。
- 跳转源对象。
- 查看 Qdrant payload 摘要。

### 状态

| 状态 | 表现 |
|---|---|
| not_built | 空状态提示构建索引 |
| building | 显示运行中，不清空旧索引 |
| ready | 显示最近成功时间 |
| stale | 显示新鲜度风险和重建入口 |
| failed | 显示失败原因，最近成功索引继续可用 |

## 可信检索

### 页面目标

在当前项目内检索代码、知识、接口、SQL 和调用链，并支持结果复用。

### 线框

```text
页面标题：可信检索

搜索区：
  [ 输入问题或关键词                                      ] [搜索]
  来源 [全部/代码/知识/接口/SQL/调用链] 状态 [全部/已确认/待复审]

结果区：
┌────────────────────────────────────────────────────────┐
│ 标题 | 来源类型 | 分数 | 状态 | 证据数                  │
│ 摘要...                                                │
│ 推荐原因...                                            │
│ [查看源对象] [复制] [加入上下文] [绑定证据] [生成知识草稿] │
└────────────────────────────────────────────────────────┘

右侧：本次检索摘要
  topScore
  resultCount
  是否低命中
```

### 结果展示规则

- 已确认知识优先突出。
- 待复审知识显示风险标识。
- 草稿知识默认不进入团队级结果，除非作者显式筛选。
- 每条结果必须可跳转源对象。

### 复用操作

| 操作 | 结果 |
|---|---|
| 复制 | 复制带来源的文本摘要 |
| 加入上下文 | 添加到 AI 研发助手候选区 |
| 绑定证据 | 打开证据绑定弹窗 |
| 生成知识草稿 | 创建 KnowledgeAsset 草稿 |

## AI 研发助手

### 页面目标

基于任务描述召回候选上下文，用户勾选、排除、排序后生成可复制上下文包。

### 线框

```text
页面标题：AI 研发助手

左侧：任务输入
  任务类型 [需求开发/缺陷修复/接口变更/重构/排障/代码评审]
  任务描述 [________________________________________]
  约束     [________________________________________]
  相关文件 [可选]
  [召回上下文]

中间：候选上下文
  筛选：来源、状态、分数
  [ ] 业务规则：订单取消规则  score 0.82
  [ ] CodeFact: OrderService.cancel()
  [ ] Endpoint: POST /orders/{id}/cancel

右侧：上下文包预览
  Markdown/纯文本切换
  引用清单
  风险提示
  [生成] [复制] [导出]
```

### 交互

- 召回候选。
- 勾选或排除上下文项。
- 拖拽或按钮排序。
- 手动添加检索结果。
- 生成 Markdown 或纯文本。
- 复制或导出。

### 风险提示

| 风险 | 展示 |
|---|---|
| 待复审知识 | 在正文和引用清单标注 |
| 引用失效 | 阻止导出或提示剔除 |
| 内容超长 | 提示删减候选 |
| 权限变化 | 导出前重新校验，失败时提示 |

## 上下文包详情

### 页面目标

查看历史上下文包、引用清单和导出审计。

### 线框

```text
页面标题：上下文包详情

基础信息
  任务类型、任务描述、创建人、创建时间、导出时间

内容预览
  Markdown / 纯文本

引用清单
┌────────────┬────────────┬────────┬────────┬────────┐
│ 来源类型    │ 标题        │ 状态   │ 分数   │ 操作   │
└────────────┴────────────┴────────┴────────┴────────┘

操作：[复制] [再次导出] [基于此包新建任务]
```

### 审计

每次导出记录导出人、时间、引用清单和脱敏摘要。
