面向项目空间的研发智能控制台

这是一套低保真但可评审的前端页面原型图，覆盖登录、项目管理、代码智能、知识资产、统一索引、 可信检索、AI 上下文包、变更影响、治理复审和审计。每张图保留真实业务控件、状态、列表和跳转关系。

项目接入->代码事实->Neo4j-> KnowledgeAsset->Qdrant->ContextPackage-> ReviewTask
贯穿页面的项目状态轨道
1当前项目、分支、快照始终可见
2分析、索引、图关系、导出状态可追踪
3所有结果能跳转源对象和审计记录
4任务失败保留上一份可用结果
