# 综合研发智能平台页面原型索引

本目录基于 `docs/superpowers/requirements/` 和 `docs/superpowers/designs/` 输出页面原型。原型采用文本线框方式，覆盖组织层、项目空间、代码智能、知识资产、检索上下文、变更影响、治理复审和审计页面。

页面原型回答“页面上有什么、怎么操作、状态如何呈现、从哪里来去哪里”。不描述具体接口实现，接口和对象字段以 `docs/superpowers/designs/` 为准。

## 原型约定

- 页面默认基于 Vue 3、Element Plus 和项目空间布局实现。
- 所有项目内页面顶部必须显示当前项目、分支、最近快照和关键状态。
- 操作按钮是否展示由前端角色控制，但最终权限必须由后端校验。
- 列表页默认支持分页、筛选、排序和空状态。
- 任务型页面必须展示任务状态、失败原因和重试入口。
- 检索结果、上下文引用、证据、ReviewTask 必须能跳转到源对象。

## 关键用户路径

### MVP 主路径

```text
登录
  -> 项目列表
  -> 项目总览
  -> 同步仓库/代码分析
  -> 知识资产绑定证据
  -> 构建索引
  -> 可信检索
  -> 加入上下文包
  -> AI 研发助手生成上下文包
  -> 导出并记录审计
```

### 第二切片路径

```text
输入文件列表或 diff
  -> 变更影响分析
  -> 命中知识或证据
  -> 生成待复审候选
  -> ReviewTask 处理
  -> 回写知识/Evidence
  -> 标记索引 stale
  -> 治理看板展示闭环结果
```
